Oque são Medias Queries

As media queries são uma funcionalidade fundamental do CSS que permite criar layouts responsivos, adaptando o design de um site ou aplicação às características do dispositivo do usuário, como largura, altura, orientação da tela e resolução. Introduzidas no CSS3, elas possibilitam que os desenvolvedores definam regras específicas para diferentes condições, garantindo uma experiência de usuário otimizada em diversos dispositivos.


Estrutura básica de uma media query

A estrutura segue o padrão abaixo:

@media (condição) { /* Regras CSS específicas */ }

Por exemplo, para alterar o estilo de uma página em dispositivos com largura de tela máxima de 768px (comum para tablets), pode-se usar:

@media (max-width: 768px) { body { font-size: 14px; } }


Tipos de condições comuns

Largura e altura da tela:

  1. min-width, max-width, min-height, max-height

    2. Orientação:

  2. orientation: portrait (vertical)
  3. orientation: landscape (horizontal)

    4. Resolução:

  4. min-resolution, max-resolution (em dpi ou dppx)

Vantagens

  • Melhor experiência de usuário ao adaptar o conteúdo ao dispositivo.
  • Facilita a manutenção do código, separando regras específicas.
  • Reduz a necessidade de criar sites ou aplicativos distintos para diferentes dispositivos.

Práticas recomendadas

  • Use breakpoints baseados no conteúdo, não apenas nos dispositivos.
  • Adote uma abordagem de "design mobile-first", começando com regras básicas para dispositivos menores e expandindo para dispositivos maiores com media queries.
